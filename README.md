###DevFolio
Welcome to DevFolio, a portfolio website showcasing the talents and projects of full stack developer Morgan Freeman. With its clean, user-friendly interface and robust navigation capabilities, DevFolio ensures a smooth and enjoyable browsing experience for visitors.

###Technologies Used
HTML
CSS
Bootstrap
Font Awesome
Media Queries

###Key Features

Responsive Design: DevFolio is designed to look spectacular on any screen size, providing a consistent and enjoyable user experience whether on desktops, tablets, or smartphones.

User-friendly Interface: The interface is intuitive, making it easy for visitors to navigate Morgan's portfolio and explore showcased projects.

Modern Design: Utilizing Bootstrap and CSS, DevFolio features a modern aesthetic while maintaining functionality and accessibility.
